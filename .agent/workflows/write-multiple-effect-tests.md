# Write Effect Test

## Goal

Write multiple integration tests, one per battle effect, as instructed by the user.

## Steps

You will execute /write-effect-test for multiple battle effects. The only difference is that all requirements MUST be received up front.

1. Gather requirements for ALL effects up front (Step 1 of /write-effect-test) for all effects.
2. Execute all other steps of /write-effect-test for each effect, based on the requirements given.
